# Dasonian Arrow Navigation must be right-handed

- Status: Accepted

## Context and Problem Statement

- By Dasonian methodology, the right hand is for movement, the left hand is for modification.
- Therefore, all arrow navigation capabilities must be constrained to keys hit by the right hand in QWERTY.
- Mouse keybinds, therefore, automatically fulfill this requirement
- One exception is the use of spacebar, for a prolonged jump, mimicing the default behaviour. Not technically a violation since spacebar can be hit with the right thumb.

## Decision Drivers

- Consistency with Dasonian methodology

## How to Check

- Check all Dasonian nav and arrow navigation is done with keys reachable via the right hand
